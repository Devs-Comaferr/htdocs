# Web para los Comerciales

Esta web obtiene la información para los clientes comerciales y la muestra. La idea es complementar a la aplicación de 'Firma de albaranes' de Área Binaria.

## Proyecto en PHP

El proyecto se ha creado en PHP de cara a poder reutilizar un viejo servidor de la empresa. La versión de PHP utilizada es la 8.3.1.
Se está llevando a cabo un proceso de actualización de todos los archivos para que funcionen correctamente en la versión nueva.
